Project: Deploy Static Website

Name: Sanjay Varatharajan

Description:
This project demonstrates deployment of a static website containing HTML, CSS, and JavaScript files.

Initially, the project was intended to be deployed using AWS services such as Amazon S3 and Amazon CloudFront. However, the AWS account was newly created and required additional verification time before services became accessible.

Due to the project deadline and the temporary activation delay on AWS, the static website was deployed using GitHub Pages as an alternative hosting platform.

CSS
JavaScript
